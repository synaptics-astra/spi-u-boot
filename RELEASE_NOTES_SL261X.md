Astra U-Boot 2025.01 Release notes for Klamath RDK boards

# 1. Information
====================

Based on U-Boot mainline version: U-Boot-2025.01

# 2. Change Log
====================
## U-Boot-2025.01-Astra-v1.0.1
--------------------
General:
	- Support ethernet
	- Support environment saving on SPI flash
	- Optimize initialization time of eMMC boot partition 

## U-Boot-2025.01-Astra-v1.0.0
--------------------
General:
	- Support USB2.0 host on type-A port
	- Support xSPI Flash
	- Support eMMC
	- Support eMMC Image upgrading from USB disk
		- Support sparse image slices
